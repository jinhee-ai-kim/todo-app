# 📝 Todo App

구글 로그인 기반의 Todo App입니다.

**[→ 데모 보기](https://jinhee-ai-kim.github.io/todo-app/)**

---

## ✨ 주요 기능

- **칸반 보드** — Todo / In Progress / Done 3단계 관리, 드래그 앤 드롭 지원
- **카테고리(Subject)** — 카테고리별 분류, 표시/숨김, 이름 변경, 삭제
- **목표(Goals)** — 상단에 고정 목표 태그 관리
- **캘린더** — 날짜별 할 일 조회
- **24H 스케줄** — 하루 시간표 시각화
- **아카이브** — 완료 항목 보관 및 검색
- **다크 모드** — 라이트/다크 테마 전환
- **Google 로그인** — 구글 계정으로 로그인
